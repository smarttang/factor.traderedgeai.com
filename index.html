DLL 是否支持 64 位 MT4？
当前发布版本为 Windows 32 位，适配 MetaTrader 4。MT5 或 64 位环境需单独构建或联系专业版获取适配版本。
如何参与多因子汇总？
使用 CTEAFactorSelector::Init(factorNames, mainFactorName)。单因子：Init("HedgeRegime")。双因子主备：Init("HedgeCore,VolatilityAdjustedMomentum", "HedgeCore")，第二个参数为主因子名（70% 权重）。3–8 因子投票：Init("HedgeCore,VolatilityAdjustedMomentum,AdaptiveTrendIntensity")，按 accuracy 加权。因子与组合因子可混用。
因子和因子组合有什么区别？
组合因子（如 HedgeCore）内部由多个原子因子组成，但对 API 而言与原子因子一样，都有统一 ID 和标准输出。可以单独用，也可以和任何原子因子、其他组合因子一起参与多因子组合，例如 Init("HedgeCore,VolatilityAdjustedMomentum")。
